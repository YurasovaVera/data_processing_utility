##Интерфейс программы
Программа представляет собой утилиту командной строки, которая принимает на вход следующие параметры:
--path путь к файлу с данными в формате .txt. Это обязательный параметр
    программа завершает свою работу с ошибкой, если
        файл не соответствует поддерживаемому формату
        файл по указанному пути не найден
--format - необязательный параметр, указывающий на формат вывода результатов. Поддерживаются форматы json, xml. По умолчанию стоит параметр json. Программа завершает свою работу с ошибкой, если на вход передан неподдерживаемый формат.

##Чтение входного файла построчно
Каждая строка файла должна содержать данные сущности, разделённые символом ;
Пример строки: 1;Anna;12.5

##Проверка корректности данных
Программа проверяет каждое поле:
id — должно быть положительным числом и не пустым.
Название — не пустое, допускаются только буквы и пробелы.
Числовое значение — должно быть положительным числом.

##Разделение записей на корректные и некорректные
Некорректные записи сохраняются в список ошибок с указанием причины.

##Подсчёт агрегированной статистики
Программа считает:
Общее количество записей
Количество корректных записей
Количество некорректных записей
Список ошибок с описанием причины для каждой некорректной записи c указание номера строки, которая содержит ошибку

##Сохранение результата в выходной файл
Файл создаётся в текущей директории с именем report.json или report.xml, в зависимости от выбранного формата.
